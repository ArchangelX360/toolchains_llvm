# CLAUDE.md - Project Analysis and Development Notes

## Project Overview

This is **toolchains_llvm**, a Bazel toolchain that provides LLVM/Clang compiler toolchains for cross-platform C++ builds. The project enables developers to use LLVM/Clang as their C++ compiler with Bazel build system across different platforms.

### Key Features
- Automatic OS and architecture detection
- Pre-built LLVM binary distributions
- Cross-compilation support
- Sysroot support for custom environments
- Multi-platform builds and remote execution
- Support for various LLVM versions

## Development Areas

### Windows Support Implementation
Current focus is on making the toolchain work properly on Windows, which involves:

1. **Wrapper Scripts**: Converting Unix shell scripts to Windows batch/PowerShell equivalents
2. **Path Handling**: Windows path conventions and escaping
3. **Tool Execution**: Proper invocation of LLVM tools on Windows
4. **Environment Variables**: Windows-specific environment setup

### Key Challenges Identified
Based on commit history, the main challenges seem to be:
- Path separator differences (/ vs \\)
- Quote escaping in Windows command line
- Script format decisions (batch vs PowerShell)
- Absolute path resolution on Windows
